# 📄 SQL Scripts

본 폴더에는  
배송 지연이 고객 만족도에 미치는 영향을 분석하기 위해 작성한  
SQL 스크립트를 **분석 단계별로 정리**하였습니다.

각 스크립트는  
데이터 전처리 → 핵심 지표 분석 → 지역별 비교 흐름으로 구성되어 있습니다.

---

## 📌 스크립트 구성

### 01_create_analysis_base.sql
🧱 분석에 사용한 **주문 단위 베이스 테이블 또는 뷰**를 생성하는 스크립트입니다.  
배송 지연 여부와 지연 일수를 계산하여 이후 분석의 기준 데이터를 만듭니다.

### 02_delay_vs_review.sql
⭐ 배송 지연 여부에 따른 **평균 고객 리뷰 점수**를 비교합니다.  
배송 지연이 고객 만족도에 미치는 영향을 1차적으로 확인합니다.

### 03_delay_bucket_analysis.sql
⏱️ 배송 지연 일수를 구간별로 분류하여  
**지연 기간이 길어질수록 고객 만족도가 어떻게 변화하는지** 분석합니다.

### 04_region_delay_analysis.sql
🗺️ 고객 지역 기준으로  
**배송 지연율과 평균 리뷰 점수의 차이**를 비교합니다.  
배송 성과 개선이 필요한 주요 지역을 식별하기 위한 분석입니다.

---

📎 각 SQL 스크립트의 실행 결과는  
Tableau 대시보드 시각화 및 인사이트 도출에 활용되었습니다.
