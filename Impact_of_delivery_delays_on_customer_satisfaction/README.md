# 📦 배송 지연과 고객 경험의 관계 분석

## 1. 프로젝트 개요

이 프로젝트는 실제 이커머스 주문 데이터를 활용하여
배송 지연 여부 및 지연 기간이 고객 만족도에 미치는 영향을 분석한 데이터 분석 프로젝트입니다.

배송 지연이 발생한 주문에서 고객 리뷰 점수가 어떻게 변화하는지 확인하고,
지역별 및 물류 단계별 지연 패턴을 통해 운영 개선에 활용 가능한 인사이트를 도출하는 것을 목표로 합니다.

⸻

## 2. 사용 데이터
	•	데이터 출처: Kaggle – Brazilian E-Commerce Public Dataset (Olist)
	•	분석 단위: 주문(order) 단위
	•	주요 테이블:
	•	orders
	•	order_reviews
	•	customers
	•	sellers
	•	geolocation
	•	order_items
	•	payments
	•	products

⸻

## 3. 주요 분석 질문
	•	배송 지연이 발생한 주문은 고객 만족도가 실제로 낮은가?
	•	배송 지연 기간이 길어질수록 고객 만족도는 어떻게 변화하는가?
	•	지역별로 배송 지연율과 고객 만족도에 차이가 존재하는가?
	•	배송 지연은 물류 과정의 어느 단계에서 주로 발생하는가?

⸻

## 4. 분석 결과 요약
	•	배송 지연이 발생한 주문은 정시 배송 주문 대비 평균 리뷰 점수가 크게 낮게 나타남
	•	배송 지연 기간이 길어질수록 고객 만족도는 단계적으로 감소
	•	일부 지역에서 배송 지연율과 낮은 만족도가 동시에 관찰됨
	•	배송 단계에서 지연이 누적되는 구조적 패턴 확인
