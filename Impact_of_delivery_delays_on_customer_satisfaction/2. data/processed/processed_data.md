# Processed Data

## 분석 단위
본 프로젝트의 분석 단위는 주문(order) 단위입니다.
각 행은 하나의 주문을 의미하며,
배송 지연 여부, 지연 일수, 고객 지역, 리뷰 점수를 기준으로 분석을 수행하였습니다. 

## 주요 전처리 기준

### 배송 관련 날짜 처리
배송 일정 관련 컬럼에는 빈 문자열('')과 NULL 값이 혼재되어 있어,
분석에 앞서 빈 문자열을 NULL로 변환하고 날짜 형식을 통일하였습니다.

또한, 배송 지연 여부를 명확하게 정의하기 위하여 실제 배송일과 예상 배송일이 모두 존재하는 주문만을 분석 대상으로 포함하였습니다. 

### 리뷰 데이터 처리
리뷰 데이터는 주문당 여러 개의 레코드가 존재할 수 있어,
각 주문에 대해 가장 최근에 작성된 리뷰를 대표 리뷰로 선정하였습니다.

리뷰 점수가 존재하지 않는 주문은
고객 만족도 분석에서는 제외하되,
배송 지연 비율 계산에는 포함하였습니다.

## 파생 컬럼 및 파생 개념

### 배송 지연 관련 파생 컬럼

- is_delayed  
  실제 배송일이 예상 배송일보다 늦은 경우 1,
  그렇지 않은 경우 0으로 정의하였습니다.

- delay_days  
  실제 배송일과 예상 배송일 간의 차이를 일 단위로 계산하였습니다.
  음수 값은 예상보다 빠르게 배송된 주문을 의미합니다.

### 배송 완료 주문 필터링

배송 지연 여부를 명확히 정의하기 위해 실제 배송 완료일과 예상 배송일이 모두 존재하는 주문만 분석 대상에 포함하였습니다.
이에 따라 분석 데이터는 배송 완료 주문으로 한정됩니다.

### 리뷰 데이터 파생 기준

주문당 여러 개의 리뷰가 존재할 수 있으므로 각 주문에 대해 가장 최근에 작성된 리뷰를 고객 만족도의 대표값으로 사용하였습니다.

### 배송 지연 구간화 기준

배송 지연 일수는 분석 편의를 위해 다음과 같이 구간화하여 활용하였습니다.

- 정상 또는 조기 배송
- 1–3일 지연 (경미한 지연)
- 4–7일 지연 (중간 지연)
- 8일 이상 지연 (심각한 지연)

### 지역 단위 파생 지표

지역별 분석에서는 다음과 같은 파생 지표를 사용하였습니다.

- 전체 주문 수
- 배송 지연 주문 수
- 배송 지연율
- 평균 리뷰 점수

## 분석 제외 데이터
배송 일정 정보가 불완전한 주문과 배송 완료 이전 상태의 주문은 배송 지연 분석의 정확도를 위해 분석 대상에서 제외하였습니다.
