# Tableau 대시보드

본 프로젝트의 최종 분석 결과는 Tableau Public 대시보드를 통해 시각화하였습니다.

## 📊 대시보드 개요

**대시보드명**  
배송 지연과 고객 경험의 관계 분석 대시보드

**주요 내용**
- 배송 지연 여부 및 지연 기간에 따른 고객 만족도 비교
- 지역별 배송 지연율 및 고객 만족도 분포
- 배송 단계별 실제 배송 소요 기간 분석
- 핵심 지표(KPI)를 통한 전체 배송 성과 요약
