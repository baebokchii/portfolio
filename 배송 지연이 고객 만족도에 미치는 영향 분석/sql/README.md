# SQL Scripts

본 폴더에는 배송 지연과 고객 만족도 분석을 위해 작성한
SQL 스크립트가 단계별로 정리되어 있습니다.

- 01_create_analysis_base.sql  
  분석에 사용한 주문 단위 베이스 테이블(또는 뷰)을 생성하는 스크립트

- 02_delay_vs_review.sql  
  배송 지연 여부에 따른 평균 리뷰 점수 비교

- 03_delay_bucket_analysis.sql  
  배송 지연 일수 구간별 고객 만족도 분석

- 04_region_delay_analysis.sql  
  고객 지역별 배송 지연율 및 평균 리뷰 점수 분석
