# 배송 지연이 고객 만족도에 미치는 영향 분석

## 프로젝트 요약
본 프로젝트는 브라질 이커머스 플랫폼 Olist의 주문 데이터를 활용하여  
배송 지연이 고객 만족도(리뷰 점수)에 미치는 영향을 분석한 데이터 분석 프로젝트입니다.

주문 단위 데이터를 기준으로 배송 지연 여부, 지연 기간, 고객 지역 정보를 결합하여  
배송 성과와 고객 경험 간의 관계를 정량적으로 확인하고,  
운영 관점에서 활용 가능한 시사점을 도출하는 것을 목표로 하였습니다.

---

## 핵심 분석 질문
- 배송 지연이 발생한 주문은 고객 만족도가 실제로 낮은가?
- 배송 지연 기간이 길어질수록 고객 만족도는 어떻게 변화하는가?
- 지역별로 배송 지연율과 고객 만족도에 차이가 존재하는가?

---

## 주요 분석 결과 요약

### 1. 배송 지연 여부와 고객 만족도
배송 지연이 발생한 주문의 평균 리뷰 점수는  
정상 배송 주문 대비 약 **1.7점 낮게** 나타났습니다.

이는 배송 지연이 고객 만족도를 단순히 소폭 저하시키는 요인이 아니라,  
고객 경험에 구조적인 영향을 미치는 핵심 요소임을 시사합니다.

### 2. 배송 지연 기간과 고객 만족도
배송 지연 기간이 길어질수록 평균 리뷰 점수가 단계적으로 하락하는 패턴이 확인되었습니다.

- 정상 또는 조기 배송: 평균 4.29점  
- 1–3일 지연: 평균 3.29점  
- 4–7일 지연: 평균 2.10점  
- 8일 이상 지연: 평균 1.70점  

특히 1–3일의 비교적 짧은 지연에서도 고객 만족도가 크게 하락하는 경향이 나타났습니다.

### 3. 지역별 배송 지연율과 고객 만족도
지역별 분석 결과, 배송 지연율이 높은 지역에서 평균 리뷰 점수가 낮은 경향을 보였습니다.  
반면 주문 수가 많고 물류 인프라가 상대적으로 안정적인 지역에서는  
배송 지연율이 낮고 고객 만족도가 높게 유지되는 패턴이 확인되었습니다.

이는 배송 성과의 지역별 편차가 고객 경험 차이로 이어질 가능성을 시사합니다.

## 사용 데이터
- Brazilian E-Commerce Public Dataset by Olist (Kaggle 공개 데이터)
- 기간: 2016년 ~ 2018년
- 분석 단위: 주문(order) 단위

※ GitHub 저장소 용량 제한으로 인해 일부 컬럼은 raw 데이터에서 제거하였으며,  
분석 목적과 직접적으로 관련된 핵심 데이터는 모두 유지하였습니다.  
자세한 내용은 `data/raw/README.md`를 참고해 주세요.

## 분석 방법 개요
- 배송 완료일과 예상 배송일을 기준으로 배송 지연 여부 및 지연 일수 정의
- 주문당 대표 리뷰 점수 선정
- 배송 지연 여부 및 지연 기간 구간별 고객 만족도 비교
- 고객 지역별 배송 지연율 및 평균 리뷰 점수 분석

분석 과정에서 날짜 형식 불일치, 결측치, 성능 이슈를 인지하고  
SQL 뷰 및 분석용 테이블을 활용하여 이를 해결하였습니다.
